Canvas Tags

Definition

According to w3schools.com, the HTML canvas element is used to draw graphics using JavaScript.
The canvas element is only a container for graphics.
Script tags are required to draw the graphics.
Canvas has several methods for drawing paths, boxes, circles, text, and adding images.
The canvas element must be acompanied by a id attribute, so it can be referred to by JavaScript.
The height and width attribute is needed to define the size of the canvas withing the HTML page.
Multiple canvas elements may be applied to one HTML page.


This is an example of using canvas tags to add some flavor to a web page.

                

                    canvas id="canvas" width="600" height="400"/>
                        script>
                        var canvas = document.getElementById("canvas");
                        var c = canvas.getContext("2d");

                        c.font = '60pt Arial';// sets the size and font used
                        c.fillStyle = 'hotpink';// sets fill color for text
                        c.strokeStyle = 'black';//sets border color

                        var x = 20,
                            y = 60,
                            text = 'CIT261 is Fun!!!';
                        c.fillText(text , x, y);
                        c.strokeText(text, x, y);
                    /script>
