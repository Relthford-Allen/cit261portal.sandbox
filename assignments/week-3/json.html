JSON Parse, Stringify

Definition:

JSON is an acronym for Javascript Object Notation. It is a standard that uses text to represent Javascript objects. It is used in server-client data transmission in web applications eg., sending data from server to client so it can be displayed in a browser, or vice versa.

Here are 2 free sites available for assisting with generating JSON objects and validating them. To generate a JSON object you can use filltext.com. Here is a validator you can use for JSon. JSONlint

JSON Structure

A JSON object is a string which has similar appearance to Javascript object literal. You are able to include the same basic data types, like strings, numbers, booleans, etc. See below an example of a JSON object which was generated from www.filltext.com, once you get to know how to do it. For those interested to create this I typed the following to produce the JSON object.

http://www.filltext.com/?rows=3&pretty=true&ID={index}&fname={firstName}&lname={lastName} &business={business}a&ddress={addressObject}
